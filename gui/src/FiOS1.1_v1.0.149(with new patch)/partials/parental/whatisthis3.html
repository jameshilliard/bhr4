Create Schedule:

What's This?

To create a schedule for the Blocked Items or Allowed Items list you created above,

1. Select one or more days of the week that you want your new Parental Control Rule to be applied. Select these days by clicking on the box next to the day(s).

2. Select the times of the day this rule will be active. You can do this by either selecting the times it will be active, or the time it will not be active. Select Start and End times by choosing both the hours and the minutes with the drop down menus. Do not forget to select AM or PM by clicking on the buttons next to them.

Here are two examples:

Example 1: If you wanted to only allow a rule to be active on Mondays from 9:00 AM to 4:00 PM, you would select Monday and click on Rule will be active at the scheduled time and then enter 9:00 AM for the Start time and 4:00 PM for the End time.

Example 2: If you wanted to allow a rule to be active ALL the time except Friday from 5:00 PM to 9:00 PM, you would select Friday above and then select Rule will be inactive at the scheduled time and then enter 5:00 PM in the Start time and 9:00 PM in the End time.
