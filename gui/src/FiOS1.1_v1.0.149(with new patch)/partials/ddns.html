Failed - Settings

One or more of the settings specified by the user is invalid and causing an error to be returned by the DDNS service. This breaks down into four different possible specific errors:

  • Incorrect username & password
  • Hostname is not one of those that has been registered in the account
  • Hostname is not a correct, fully-qualified domain name
  • User has specified an option that is only for paid accounts, and this is not a paid account
