Parental Controls

The Router provides basic Parental Controls that allow you to create a list of website addresses and keywords embedded in website addresses that will limit the computer user's Internet access. Simply follow the 3 Steps below and click the Apply button to set up your Parental Controls.

Note: While these basic Parental Controls are a great way to limit access to particular sites, there are other computer software applications that provide computer Monitoring and computer Content Cleanup. Monitoring involves keeping records of the computer user's activity for later review. Content Cleanup involves scanning the actual content of websites, emails, and attachments for specific words to block or for spyware, popups, adware, etc.

Step 1. Select the Networked Computer / Device for this Allow or Block Rule

What's this?

The Networked Computer / Device list contains all such devices recognized on your home network, by name or by IP address if not named. You may select one, more than one, or all such devices to be added to the list of Selected Devices for the Rule you are creating. You may need to scroll down this list to see all of the available such devices.

To Add computer(s)/device(s) into the Selected Devices list, click on the name of the computer/device you wish to add and then click the Add button. You may add more than one at a time.

To Remove a computer/device from the Selected Devices list, click on the name of the computer/device you wish to remove then click the Remove button. You may remove more than one at a time. Note that this action WILL NOT remove the computer/device from your computer, only from the list of Selected Devices.

Note: You can select only specified devices, not specific accounts on these devices.
